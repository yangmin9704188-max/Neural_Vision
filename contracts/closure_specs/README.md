# Closure Specs

- Directory for per-step or per-feature closure specifications.
- Canonical protocol: `contracts/feature_closure_protocol_v1.md`.
- Template: `contracts/templates/closure_spec_template_v1.md`.

Naming rule:
- Preferred: `contracts/closure_specs/<module>/<STEP_ID>.closure_spec.md`
- Process-level example: `contracts/closure_specs/body/measurement_engine.closure_spec.md`
